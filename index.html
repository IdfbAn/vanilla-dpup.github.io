Vanilla Dpup 9.2.x (Stable)

This is a stable, modern and polished Puppy Linux flavor based on Debian Bullseye packages, with weekly stability and security fixes.

It's small, but features apt-get, classic themes, Bluetooth audio support, DNS caching, ad blocking, shell completion, man pages, keyboard shortcuts, and much more, in the traditional Puppy format.

Vanilla Dpup 9.3.x (Coming Soon)

This brings some of the new features that will appear on Vanilla Dpup 10.0.x, on top of the stable and established platform of 9.2.x.

It features overlay instead of aufs, an installer with UEFI support, simple installation of out-of-tree drivers, built-in multi-language support, more desktop customization options and more built-in applications.

Vanilla Dpup 10.x (Unstable)

This version is in development and no new major features are planned. It tracks Debian Bookworm, improves Bluetooth audio support by replacing PulseAudio with PipeWire, adds Landlock-based sandboxing for internet-facing applications and adds a Wayland flavor.
